Node

Node is a runtime enviroment for executing JS Before node we used JS only to build applcations that runs inside a browser. So every browser has what we call a JS-engine that takes our JS-code and convert it to code that the computer can understand. In 2009 Ryan Dahl, the creator of node, thought it would be great to execute JS outisde of a browser. So he took googles v8 engine, which is the fastes JS-engine and embeded it inside a c++ program and called that program node.

Express

Express is the most common framework for building web applications and web server on top of node. It provides mechanisms to: Write handlers for requests with different HTTP verbs at different URL paths (routes). Integrate with "view" rendering engines in order to generate responses by inserting data into templates. Set common web application settings like the port to use for connecting, and the location of templates that are used for rendering the response.
